## 🎯 Quick Start Commands

```bash
# Install
npm install

# Run tests
npm test                    # All tests
npm run test:headed         # With browser visible
npm run test:debug          # Debug mode
npm run test:report         # View HTML report

# By browser
npm run test:chrome
npm run test:firefox
npm run test:webkit
```
